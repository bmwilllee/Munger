# 芒格式股票价值分析工具

一个本地运行的公司股票价值分析工作台。输入股票代码后，工具会拉取真实公开行情与财务数据，并从查理·芒格偏好的长期价值投资视角生成可视化分析。

## 启动

```bash
node server.js
```

打开：

```text
http://127.0.0.1:4173
```

## 安装为本地常驻服务

macOS 下运行：

```bash
chmod +x install-local-service.sh uninstall-local-service.sh
./install-local-service.sh
```

安装后会在登录时自动启动，并保持运行。卸载：

```bash
./uninstall-local-service.sh
```

## 分析框架

工具把公司拆成四个维度评分：

- 生意质量：ROIC、ROE、经营利润率、自由现金流率
- 持久性：营收增长、自由现金流增长、负债压力、波动风险
- 管理层与资本配置：回购、杠杆、现金转化质量
- 估值与安全边际：保守 DCF、P/E、P/FCF、市场隐含增长

新增的深度判断模块：

- 护城河证据：资本效率、定价权、可复投空间、抗脆弱性，以及对应反证风险
- 财报质量：现金利润转换、自由现金流连续性、利润率稳定性、股本稀释、净债务压力
- 估值情景：熊市、基准、乐观 DCF，以及 30% / 15% 安全边际买入区间
- 投资备忘录：把生意质量、财报可信度、价格判断和下一步核查事项合并成决策摘要

## 数据说明

当前免 key 模式使用：

- SEC EDGAR Company Facts：美国上市公司的真实年报财务数据
- Stooq：最新行情报价，可能存在交易所延迟

工具不会再回退到演示数据。真实数据不可用时，页面会直接提示失败原因。

本工具用于研究和决策辅助，不构成投资建议。
